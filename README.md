# simonEcommerce proyecto basico de comercio Electronico con React js


## Objectivos de Implementacion:

* Cargar productos de una base de datos(firebase)
* Filtrar productos por Categoria
* Ver detalles de cada producto
* Agregar/remover productos al carrito desde:

  * Listado inicial
  * Detalle de producto
  * Carrito
  
* Agregar un carrito de compras
* Agregar detalle de carrito de compras

  * tabla con cantidad de items
  * nombre* cantidad
  * precio unitario
  
* Mostrar total global de items en el carrito de compras
* Permitir en el carrito de compras quitar/agregar productos (se agregaron btns de agregar/quitar y remover todo el producto)
* Vaciar carrito de compras
* Generar ID de order de compra por medio de formulario en el carrito


## Para este proyecto se usaron las siguientes tecnologias:
* HTML
* CSS
* React (mediante el uso de vite)

Ademas se usaron la siguientes librerias:
